# Assets Directory

This directory contains brand assets for the EchoKey platform.

## Files

- **logo.png** - Main EchoKey logo (metallic text with glow effect)

## Usage

The logo is used throughout the application:
- Header navigation
- Home page hero section
- Documentation

## Branding Guidelines

- Background: Dark (#0A0F2C recommended)
- Primary Color: #5C6AFF (purple-blue glow)
- Accent Color: #9CE7FF (cyan)
- The logo should maintain sufficient contrast against dark backgrounds
